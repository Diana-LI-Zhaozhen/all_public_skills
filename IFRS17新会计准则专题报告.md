# IFRS 17 / IFRS 9 新保险会计准则

> \*\*标注规则\*\*：🔷 标注精算/财务模型；📐 标注计算公式

\---

## 第一章 业务实质：保险合同的定义、识别与分组

### 1.1 保险合同的适用范围与识别

IFRS 17于2017年5月由IASB发布，取代IFRS 4，适用于：

* 所有**签发的保险合同**和**再保险合同**
* 同时签发保险合同的实体所签发的**具有自由参与特征的投资合同**
* 固定收费服务合同（主体可选择按IFRS 17或IFRS 15处理）
* 财务担保合同（若主体此前明确声明将其作为保险合同处理）

**排除**：作为投保人的保险合同（再保险除外）。

#### 合同成分拆分规则

|旧准则（IFRS 4）|新准则（IFRS 17）|
|-|-|
|先拆分混合保险合同，再对无法拆分的合同做重大风险测试|**先逐合同做重大风险测试**，通过后再拆分成分|

拆分后的成分处理：

1. **保险成分 + 不可明确区分的投资成分** → 适用 IFRS 17
2. **可明确区分的投资成分** → 适用金融工具准则（IFRS 9）
3. **嵌入衍生工具** → 单独处理
4. **可明确区分的商品/非保险合同服务** → 适用收入准则（IFRS 15）

### 1.2 保险合同分组逻辑

🔷 **分组模型（Grouping Model）**：一个**组合（Portfolio）**（受类似风险影响并作为单一池共同管理的合同）必须在初始确认时拆分为**3组**：

|组别|含义|CSM处理|
|-|-|-|
|**第1组**|亏损合同（初始确认时即为亏损）|CSM = 0，差额立即计入损益|
|**第2组**|无重大变为亏损风险的合同|CSM > 0，正常摊销|
|**第3组**|剩余盈利合同|CSM > 0，正常摊销|

**关键规则**：

* 签发时间相差超过1年的合同**不能归入同一组**
* 窄豁免：法律或监管约束对定价的影响可免于正常分组规则
* CSM通过\*\*覆盖单位（Coverage Units）\*\*在每个期间释放至损益

### 1.3 直接参与特征保险合同定义（VFA适用前提）

合同在初始确认时须**同时满足三项标准**才能使用VFA（参照IFRS 17 ¶B101、¶BC238）：

|标准|内容|
|-|-|
|**VFA标准I**|合同条款规定保单持有人参与**明确识别的基础项目池**的份额|
|**VFA标准II**|主体预期向保单持有人支付的金额等于基础项目公允价值回报的**重大份额**|
|**VFA标准III**|主体预期向保单持有人支付金额变动的**重大比例**随基础项目公允价值变动而变化|

> \*\*判断要点\*\*：IASB未提供"重大"（substantial）的定量定义，需主体自行评估并获得审计师确认；基础项目池无需主体实际持有；不同司法管辖区的判断结果可能不同。

## 第二章 会计项目：核心计量要素及计算公式

### 2.1 三大计量模型概览

🔷 **通用模型（GMM/BBA）**：默认模型，适用于所有非直接分红合同  
🔷 **浮动收费法（VFA）**：适用于通过3项标准的直接参与特征合同（分红险、投连险）  
🔷 **保费分配法（PAA）**：适用于覆盖期≤1年的短期合同

|维度|GMM|VFA|PAA|
|-|-|-|-|
|**适用合同**|非分红保单、长期保障型|直接分红保单、投连险|短期一般保险|
|**CSM利息**|按初始锁定利率明确计提|隐含在可变费用变动中|无需详细CSM计算|
|**底层资产联动**|无|损益变动与底层资产处理镜像|无|
|**折现率变动**|可选OCI|可选OCI|可选OCI|
|**首日利润**|确认为CSM（非负），无首日利润|CSM含浮动收费|无详细CSM|

### 2.2 核心计量要素定义与公式

#### 2.2.1 履约现金流（Fulfillment Cash Flows, FCF）

📐 **FCF计算公式**：

$$FCF = \\sum\_{t} \\left( \\text{概率加权现金流}\_t \\times \\text{折现因子}\_t \\right) + RA$$

展开为：

$$\\boxed{FCF = BEL + RA}$$

其中：

* **BEL**（最佳估计负债）= 折现后概率加权最佳估计现金流
* **RA**（风险调整）= 反映主体因承担不确定性而要求的补偿

FCF包含：未来保费（流入）、赔款和费用（流出）、投资组成部分的扣除。

#### 2.2.2 风险调整（Risk Adjustment, RA）

RA量化了"确定负债"与"不确定负债"之间的价值差异，与折现现金流分离列示。

**计算方法**（新加坡实践）：

* 多数公司采用**置信水平法**，披露区间多为75%–85%百分位
* 少数公司（如Manulife）采用**边际法**
* 一般保险公司：71%校准概率为75%或接近75%

#### 2.2.3 合同服务边际（Contractual Service Margin, CSM）

📐 **初始确认公式**：

$$\\boxed{CSM\_{初始} = \\text{保费（扣除投资成分）} - FCF}$$

或展开为：

$$CSM\_{初始} = \\text{预期现金流入现值} - \\text{预期现金流出现值} - RA$$

* 若结果**为正** → 确认CSM
* 若结果**为负** → 合同组归类为亏损，**不确认CSM**，差额立即计入损益

📐 **GMM后续计量公式**：

$$\\boxed{CSM\_{EOY}^{GMM} = CSM\_{BOY} + CSM\_{BOY} \\times r\_{初始} - 摊销 + \\Delta 非经济假设}$$

其中：

* $r\_{初始}$ = 初始确认时的锁定折现率
* 摊销 = $(CSM\_{BOY} + 利息计提) \\times \\frac{当期覆盖单元}{总覆盖单元}$
* 非经济假设变动（死亡率、发病率等）由CSM吸收

📐 **VFA后续计量公式**：

$$\\boxed{CSM\_{EOY}^{VFA} = CSM\_{BOY} + \\Delta 可变费用 - 摊销}$$

其中：

* **可变费用变动** = 基础项目公允价值变动 − FCF展开成本
* **可变费用** = 主体享有的基础项目公允价值份额 − 不随基础项目变动的履约现金流
* VFA下**不单独计提CSM利息**，利息隐含在可变费用变动中
* VFA下CSM计息使用**当期即期利率**（非锁定利率）

#### 2.2.4 覆盖单元（Coverage Units）

📐 **覆盖单元公式**：

$$\\text{覆盖单元}\_t = NOP\_t \\times SA\_t$$

其中NOP = 有效保单数量，SA = 保额（含账户价值）。

每期CSM摊销额 = CSM待摊余额 × 当期覆盖单元 / 总覆盖单元。

#### 2.2.5 保险合同负债

📐 **总负债公式**：

$$\\boxed{\\text{保险合同负债} = FCF + CSM = BEL + RA + CSM}$$

### 2.3 折现率的确定

IFRS 17规定采用两种方法获取反映保险负债特征的折现率：

|方法|逻辑|
|-|-|
|**自上而下（Top-down）**|从可观察的市场收益率出发，扣除与非流动性保险负债无关的风险溢价|
|**自下而上（Bottom-up）**|从无风险利率出发，加入反映保险负债特征的非流动性溢价（ILA）|

**关键变化**（vs 旧准则）：

* 旧准则：750日移动平均国债收益率曲线 + 综合溢价（最高120BP），可自主调节
* 新准则：基准为即期国债利率，**取消平滑机制和自主调整空间**
* 新赋予**OCI选择权**：可将折现率变动引发的金融变动额计入OCI

**新加坡实践**（2023年12月31日新元数据）：

* 人寿公司：自下而上为主（6/8家），非流动性溢价区间2.40%–4.44%
* 一般保险：绝大多数自下而上，IP贡献极小或为0%

### 2.4 OCI选择权

|选择|效果|
|-|-|
|**计入损益（P\&L）**|折现率变动直接影响当期利润|
|**计入OCI**|折现率变动不计入当期利润，减少利润波动性；后续在合同剩余期限内逐步分摊转入承保财务损益|

**不同模型下OCI策略**：

* **GMM**：OCI是管理利润波动的核心工具
* **VFA**：较少使用OCI，更多依赖资产与负债的直接联动管理
* **PAA**：精算假设变动直接计入P\&L

## 第三章 报表勾稽：资产负债表、利润表与OCI的联动

### 3.1 资产负债表勾稽

#### 保险合同负债变动勾稽

📐 **期末保险合同负债公式**：

$$\\text{期末ICL} = \\text{期初ICL} + \\Delta\_{当期服务} + \\Delta\_{未来服务} + \\Delta\_{过去服务} + \\text{保险金融费用} - \\text{保费收入} + \\text{赔付支出} + \\text{获取费用}$$

其中：

* $\\Delta\_{当期服务}$ = CSM释放 + RA释放 + 经验偏差
* $\\Delta\_{未来服务}$ = 新合同 + 履约现金流变动 + 亏损调整
* $\\Delta\_{过去服务}$ = 未决准备金调整

#### 科目映射变化

|旧准则科目|新准则列示|变化原因|
|-|-|-|
|各类准备金（未到期/未决/寿险/长健）|合并为「保险合同负债」|负债统一计量|
|保户储金及投资款（万能险）、独立账户资产/负债（投连险）|取消，并入保险合同资产/负债|混合合同先测后拆|
|保户质押贷款|取消，纳入保险合同资产/负债|权责发生制下属于保单负债现金流|
|与保费有关的应收/应付科目|取消|新准则采用权责发生制|
|资产减值损失|拆分为「信用减值损失」「其他资产减值损失」|IFRS 9减值模型调整|

### 3.2 利润表勾稽

📐 **利润表核心勾稽**：

$$\\boxed{\\text{保险服务业绩} = \\text{保险服务收入} - \\text{保险服务费用} + \\text{摊回保险服务费用} - \\text{分出保费分摊}}$$

$$\\boxed{\\text{投资服务业绩} = \\text{总投资收益} - \\text{承保财务损益}}$$

$$\\boxed{\\text{营业利润} = \\text{保险服务业绩} + \\text{投资服务业绩} - \\text{业务及管理费} - \\text{信用减值损失等} + \\text{其他收益}}$$

$$\\boxed{\\text{净利润} = \\text{营业利润} + \\text{营业外收支} - \\text{所得税}}$$

$$\\boxed{\\text{综合收益总额} = \\text{净利润} + \\text{其他综合收益税后净额}}$$

#### 保险服务收入明细

📐 **保险服务收入公式**：

$$\\text{保险服务收入} = \\text{预计保险服务费用} + \\text{非金融风险调整变动} + \\text{CSM摊销} + \\text{保险获取现金流摊销}$$

> 对应传统"死差+费差"的利润来源。

#### 承保财务损益

$$\\text{承保财务损益} = \\text{负债计息增值} + \\text{折现率变动导致的负债变动}$$

> 对应传统"利差"的反面。若行使OCI选择权，折现率变动部分计入OCI而非此处。

### 3.3 OCI勾稽关系

📐 **其他综合收益变动公式**：

$$\\Delta OCI = \\underbrace{\\text{IFRS 9: FVOCI资产公允价值变动}}*{\\text{资产端}} + \\underbrace{\\text{IFRS 17: 计入OCI的保险合同金融变动额}}*{\\text{负债端}} + \\underbrace{\\text{分出再保险合同金融变动额}}\_{\\text{再保险端}}$$

📐 **净资产勾稽**：

$$\\text{净资产}*{EOY} = \\text{净资产}*{BOY} + \\text{净利润} + \\Delta OCI - \\text{分红等权益变动}$$

#### OCI天然错配问题

1. 资产端权益类OCI**无法与负债端形成对冲**，且FVOCI权益卖出时增值收益无法进入净利润
2. 资产端暴露利率风险的资产通常为负债端的**一半左右**，且资产久期普遍小于负债久期
3. 负债端OCI逐年平滑进净利润，资产端固收类OCI卖出时一次性释放——存在**天然错配**

### 3.4 跨准则勾稽（IFRS 9 × IFRS 17）

若资产端债权工具选择OCI指定权，负债端保险合同金融变动也需选择OCI指定权，否则出现"资产端波动进OCI、负债端波动进损益"的错配，导致利润表波动失真。

## 第五章 IFRS 17指标定义及产出

### 5.1 精算模型体系

🔷 **精算模型层级**：

```
保险合同组合（Portfolio）
  └─ 合同组（Group）：第1组/第2组/第3组
       └─ 计量模型选择：
            ├─ GMM（通用模型）→ 非分红长期合同
            ├─ VFA（浮动收费法）→ 直接分红合同
            └─ PAA（保费分配法）→ 短期合同
```

### 5.2 GMM完整计量流程与公式

📐 **Step 1：计算FCF**

$$FCF = PV(\\text{概率加权未来现金流}) + RA$$

展开现金流：

* 流入：保费
* 流出：佣金、费用、身故给付（保险成分+投资成分）、生存给付
* 净现金流（NCF）= 流入 − 流出

📐 **Step 2：计算初始CSM**

$$CSM\_{初始} = \\max(-FCF, 0)$$

📐 **Step 3：计算覆盖单元**

$$CU\_t = NOP\_t \\times SA\_t$$

📐 **Step 4：后续每期CSM滚动**

$$CSM\_t^{GMM} = CSM\_{t-1} + CSM\_{t-1} \\times r\_{lock} - \\frac{CU\_t}{\\sum CU} \\times (CSM\_{t-1} + CSM\_{t-1} \\times r\_{lock}) + \\Delta 非经济假设$$

### 5.3 VFA完整计量流程与公式

📐 **Step 1–3**：与GMM相同

📐 **Step 4：计算可变费用变动**

$$\\Delta VF\_t = \\Delta \\text{基础项目公允价值}\_t - \\Delta \\text{FCF展开成本}\_t$$

其中**可变费用**的定义：

$$VF = \\text{主体享有的基础项目公允价值份额} - \\text{不随基础项目变动的履约现金流}$$

📐 **Step 5：VFA CSM滚动**

$$\\boxed{CSM\_t^{VFA} = CSM\_{t-1} + \\Delta VF\_t - 摊销\_t}$$

### 5.4 数值案例：5年期投连产品GMM vs VFA

**产品参数**：身故给付=固定500+AV；均衡年保费=500/份；100份保单；2% AMC；RA=0；初始折现率=PHAV增长率（≈3%）

|年度|GMM CSM(EOY)|VFA CSM(EOY)|差异|
|-|-|-|-|
|初始|3,200|3,200|0|
|Yr1|2,913|3,038|+125|
|Yr2|2,472|2,810|+338|
|Yr3|1,858|2,358|+500|
|Yr4|1,047|1,467|+420|
|Yr5|0|0|0|

> \*\*核心结论\*\*：VFA的CSM始终高于GMM，因为可变费用变动（反映基础资产增值）增加了CSM，随后在后期加速摊销。VFA起到了\*\*利润缓冲器\*\*的作用。

### 5.5 亏损合同处理

|情形|处理规则|
|-|-|
|**初始确认亏损**|CSM = 0，亏损直接计入当期保险服务费用|
|**GMM后续亏损**|非经济假设变动导致FCF增加，先由CSM吸收，CSM耗尽后未吸收部分计入当期损益|
|**VFA后续亏损**|基础项目公允价值中主体份额减少额超过CSM账面价值时出现亏损，计入保险服务费用；**后续可转回**|

📐 **VFA亏损示例**：CSM账面100万，基础项目公允价值份额减少120万 → 亏损=120-100=20万计入保险服务费用；后续份额增加30万 → 可转回20万提升利润。

### 5.6 PHAV（保单持有人账户价值）计算

📐 **PHAV公式**：

$$PHAV\_{EOY} = PHAV\_{BOY} + 保费\_{BOY} - COI\_{BOY} + 投资收益\_{EOY} - AMC\_{EOY} - 身故给付来自PHAV\_{EOY}$$

## 第六章 KPI设定

### 6.1 IFRS 17核心KPI体系

#### A. 盈利性指标

|指标|📐 计算公式|说明|
|-|-|-|
|**新单合同服务边际**|$NB\\ CSM = \\Sigma 新单CSM - \\Sigma 新单亏损$|衡量新增业务预期未来利润|
|**新单CSM率**|$NB\\ CSM率 = \\frac{NB\\ CSM}{新单合同服务标准保费}$|新业务盈利能力|
|**合同服务边际率**|$CSM率 = \\frac{期末CSM}{期末保险合同负债}$|存量业务盈利厚度|
|**CSM增长率**|$CSM增长率 = \\frac{期末CSM - 期初CSM}{期初CSM}$|利润储备增长动力|
|**综合成本率**|$COR = \\frac{保险服务费用}{保险服务收入}$|财险核心指标|
|**保险负债资金成本率**|$= \\frac{承保财务损益 - 分出再保险财务损益}{平均保险合同负债 - 平均分出再保险资产}$|负债端成本|

#### B. 投资效率指标

|指标|📐 计算公式|说明|
|-|-|-|
|**资产投资收益率**|$= \\frac{投资收益 + 公允价值变动损益}{平均投资资产}$|资产端收益能力|
|**资产负债利率差**|$= 资产投资收益率 - 保险负债资金成本率$|利差收益水平|

#### C. 调整后ROE指标（德勤框架）

📐 **四种版本**：

|版本|公式|逻辑|
|-|-|-|
|**基础版**|$ROE = \\frac{承保利润}{IFRS\\ 9/17权益}$|最简单直接|
|**含RA+CSM调整版**|$ROE = \\frac{承保利润 + \\Delta RA + \\Delta CSM}{权益 + RA + CSM}$|接近SII框架|
|**含OCI调整版**|$ROE = \\frac{承保利润 + \\Delta OCI + \\Delta RA + \\Delta CSM}{权益 + RA + CSM}$|贴合SII口径，短期波动大|
|**剔除OCI版**|$ROE = \\frac{承保利润 + \\Delta RA + \\Delta CSM}{权益 + RA + CSM - OCI}$|更稳定，接近非市场一致计量|

#### D. 资本管理指标

|指标|📐 计算公式|说明|
|-|-|-|
|**调整后股东权益**|$Adj\\ SHE = IFRS权益 + 税后CSM$|新加坡市场核心KPI|
|**权益占比**|$= \\frac{权益}{权益 + 税后CSM}$|权益在总经济价值中的比例|
|**税后CSM占比**|$= \\frac{税后CSM}{权益 + 税后CSM}$|未来利润占比|
|**总CSM**|$Total\\ CSM\_T = 过渡CSM + \\Sigma NB\\ CSM + \\Sigma 波幅效应 - \\Sigma CSM释放$|存量利润储备|
|**IFRS 17新业务价值**|$NBV\_{17} = NB\\ CSM + NB\\ RA - 收购成本 - 亏损部分$|新业务综合价值|
|**IFRS 17增加值**|$VA\_{17} = 调整后营业利润 + \\Delta CSM$|期间价值创造|

#### E. 营运偏差指标

|指标|📐 计算公式|
|-|-|
|**营运偏差率**|$= \\frac{期初预计当期保险服务费用 - 当期实际赔款及费用}{期初预计当期保险服务费用}$|
|**精算假设变更影响率**|$= \\frac{调整CSM的变更}{期初CSM}$|
|**新单亏损率**|$= \\frac{新单亏损之和}{新单CSM}$|

### 6.2 KPI在管理决策中的应用

|KPI|应用场景|
|-|-|
|**CSM余额及摊销节奏**|反映未来利润释放空间；高CSM→更强稳定性、更多VFA对冲空间|
|**NB CSM**|新增业务价值核心披露指标；需理解SII NBV与NB CSM的差异|

|**有机资本生成（OCG）**|北欧/英国/荷兰头部公司用于披露指引、规划目标和分红政策|
|**调整后ROE**|分析师在同业对比中广泛使用|

## 第七章 市场对业务影响

### 7.1 利润波动影响

#### 资产端波动（IFRS 9）

* FVTPL占比大幅提升：上市险企2023年实施后FVTPL占比普遍>30%（平安从2%→24%）
* 权益工具只能划分为FVTPL或FVOCI，FVOCI指定后**不可撤销**，累计浮盈浮亏**不能转回损益表**
* ECL减值计提范围扩大、时间提前，但上市险企信用减值损失/净利润均在10%以下

**2023年Q1资产分类实际数据**：

|分类|四家上市险企平均|区间|
|-|-|-|
|权益类FVOCI占比|3.1%|新华0.4% \~ 人保5.0%|
|固收类FVOCI占比|37.4%|人保23.0% \~ 太保54.7%|
|固收类AC占比|13.0%|太保4.3% \~ 人保21.6%|

#### 负债端波动（IFRS 17）

* 寿险公司保险服务收入较原已赚保费**下滑约60%–70%**
* 保险合同负债波动性提升（折现率基于即期利率）
* 亏损合同单独披露，倒逼聚焦盈利业务

#### 权益配置波动性量化

|权益配置比例|利润波动（BBA/GMM）|利润波动（VFA）|净资产波动（BBA/GMM）|净资产波动（VFA）|
|-|-|-|-|-|
|5%|\~15%|\~8%|\~12%|\~7%|
|15%|\~25%|\~15%|\~20%|\~12%|
|25%|\~35%|\~22%|\~28%|\~18%|
|35%|\~45%|\~30%|\~35%|\~25%|

> \*\*核心结论\*\*：同等权益配置下，VFA模型利润和净资产波动显著低于GMM，分红险业务占比高的公司利润稳定性更强。

### 7.2 分账户管理策略

|账户类型|计量模型|风险承受力|配置策略核心|
|-|-|-|-|
|**传统险**|GMM|最强|资产负债端同时选用/不选用OCI；追求稳定→增AC+FVOCI权益；追求弹性→增FVTPL；**三种思路不完全兼容，需厘清管理目标**|
|**万能险**|GMM|最弱|特别储备不足时降低FVTPL权益；待储备充足后适度增加；宜放弃报表匹配，从结算收益角度管理|
|**分红险**|VFA|CSM厚度足够时较高|优先增加CSM厚度；CSM足够→可增高收益资产；CSM不足→降FVTPL比例；资产端须用公允价值计量|
|**财险**|PAA|中等|IFRS 17影响小，主要应对IFRS 9：求稳→增AC固收+FVOCI权益；求收益→增FVTPL|

### 7.3 资产配置趋势

|资产类别|调整方向|理由|
|-|-|-|
|固收资产|**增加**|现金流特征接近非参与负债；期限匹配减少久期缺口和净资产波动|
|高股息股票|**增加**|适合计入FVOCI，价格变动不影响利润，仅分红入损益|
|普通权益|**审慎**|只能选FVTPL（增波动）或FVOCI（丧失处置收益确认机会）|
|另类投资|**谨慎**|默认FVTPL，需注意与公开市场联动性|

### 7.4 产品策略调整

* 倾向推广**长期缴费保障型产品**，扩大分红险业务以适配VFA
* 亏损合同单独披露倒逼**聚焦盈利业务**，减少低质量保单扩张
* 开发精细化低波动产品，强化现金流管理
* 使用更精确的精算模型调整定价结构

## 附录B：精算模型/财务模型索引

|模型名称|类型|适用场景|核心特征|
|-|-|-|-|
|🔷 **GMM/BBA（通用模型）**|精算计量模型|非分红长期合同（默认模型）|折现现金流+RA；CSM按锁定利率计息；非经济假设变动调整CSM；可选OCI|
|🔷 **VFA（浮动收费法）**|精算计量模型|直接参与特征合同（分红险/投连险）|CSM吸收底层资产FV变动；可变费用调整；利息隐含在可变费用变动中|
|🔷 **PAA（保费分配法）**|精算计量模型|覆盖期≤1年的短期合同|简化处理，未到期负债类似未到期保费；已发生赔款用完整积木法|
|🔷 **ECL（预期信用损失模型）**|金融减值模型|IFRS 9下所有需减值测试的金融资产|三阶段（12m ECL / 终身ECL / 终身ECL+净额利息）|
|🔷 **SPPI测试模型**|金融资产分类模型|IFRS 9下债务工具分类判定|仅含本息测试，不通过则直接FVTPL|
|🔷 **覆盖单元模型**|CSM摊销模型|所有需CSM摊销的合同|$CU = NOP \\times SA$；每期前瞻性重新估计|
|🔷 **置信水平法**|RA计算方法|多数公司采用的RA量化方法|校准至指定置信区间（通常75%–85%百分位）|
|🔷 **边际法**|RA计算方法|少数公司（如Manulife）|基于资本成本或风险溢价的边际方法|
|🔷 **自上而下法**|折现率确定方法|部分公司|从市场收益率出发扣除无关风险溢价|
|🔷 **自下而上法**|折现率确定方法|新加坡多数公司|从无风险利率出发加非流动性溢价|
